# Capturas de tela reais (opcional)

A página (`docs/index.html`) já mostra representações fiéis das telas via SVG.
Para usar seus **prints reais**, salve as imagens aqui com estes nomes e elas
poderão ser referenciadas na galeria:

- `dashboard.png`
- `sensoriamento.png`
- `mapa-calor.png`
- `interpretacao-ia.png`

Sugestão: tire os prints com o app aberto (cmd+shift+4 e espaço, clique na janela).
